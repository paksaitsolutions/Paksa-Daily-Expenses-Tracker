## ðŸ›  Troubleshooting

### Common Issues:

**Charts not loading:**
- Ensure internet connection (ECharts loads from CDN)
- Refresh the page
- Check browser console for errors

**Data not saving:**
- Ensure Local Storage is enabled in browser
- Check available storage space
- Try clearing browser cache

**Mobile menu not working:**
- Ensure JavaScript is enabled
- Try refreshing the page
- Check for browser compatibility

**Export not working:**
- Ensure pop-ups are allowed
- Check download folder permissions
- Try different export format

### Performance Tips:
- **Large datasets**: Export and archive old data periodically
- **Mobile**: Close other browser tabs for better performance
- **Charts**: Refresh page if charts become unresponsive
